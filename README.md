#### i-threesixty Hire Purchase app

## Setup app in local machine

1) First Enter the folder
2) Open cmd and go to folder
3) git clone https://github.com/LasithMadu/HirePurchase.git
4) cd HirePurchase
5) npm install
6) npm start

### Important

1) Install node js
   https://nodejs.org/en/
2) Install git
   https://git-scm.com/downloads
